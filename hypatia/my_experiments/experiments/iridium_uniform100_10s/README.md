## Iridium Uniform100 10s

This experiment is a dense satellite path-flow run using:

- Iridium constellation
- 100 synthetic ground stations generated uniformly over the globe
- 1000 directed long-distance OD flows
- 10 seconds of simulation
- large TCP demands designed not to finish within the 10s run

The ground stations are generated by the shared pipeline with
`GROUND_STATION_SELECTION_MODE = "uniform_global"`, so this experiment does not
depend on top-city input files.

## Run

```bash
cd /home/xuke/tz-Hypatia/hypatia
source venv/bin/activate
export CC=gcc-9 CXX=g++-9

cd my_experiments/experiments/iridium_uniform100_10s
python run_pipeline.py --threads 4 --build
```

Use `--build` after C++ monitor changes. Otherwise it can be omitted after ns-3
has already been rebuilt.

## Traffic

The shared traffic planner generates `input/schedule.csv` automatically from
the experiment parameters:

- `TRAFFIC_FLOW_COUNT = 1000`
- `TRAFFIC_MIN_DISTANCE_KM = 7000`
- `TRAFFIC_TARGET_AVG_FLOW_SIZE_BYTES = 100_000_000`
- `TRAFFIC_MIN_FLOW_SIZE_BYTES = 50_000_000`
- `TRAFFIC_MAX_FLOW_SIZE_BYTES = 200_000_000`

Per-flow TCP progress/RTT/cwnd logging is disabled; this experiment only needs
the C++ satellite path-flow monitor output.
